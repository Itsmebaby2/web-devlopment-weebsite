Web Devlopment

Web development is the way toward building websites and applications for the web or a private network known as an intranet. Web development is not concerned about the design and layout of a website; instead, it's focused on the coding and programming that controls the website's functionality.

front-end VS back-end

" The front-end is created using a blend of technologies, for example, Hypertext Markup Language (HTML), JavaScript, and Cascading Style Sheets (CSS). Front-end developers design and create the user experience components on the website page or application, including dropdowns, buttons, menus, pages, connections, procedures, etc. "

" The back-end, also called the server-side, comprises the server that delivers information on request, the application that channels it, and the database that arranges the data. "
